## css居中常用的几种方式
 ### 行内元素水平居中
   * 方案一：
  ```html
  <!--html如下-->
    <div class="line-vertical-center-one">
            <span class="mark-item">
              这里是一个行内元素
            </span>
          </div>
   ```
   ```css
    /*css代码段*/
    .line-vertical-center-one{
      text-align: center;
    }
```
这里主要是使用text-align:center;是元素水平居中
